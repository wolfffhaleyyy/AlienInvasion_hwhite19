# Alien Invasion

## Overview

Alien Invasion is a Python game built with **Pygame** as a way to learn the fundamentals of game development. Inspired by classic arcade games like **Space Invaders** and **Galaga**, this project recreates the core gameplay while providing hands-on experience with object-oriented programming and game mechanics.

The goal of this project is not only to build a fun arcade-style game, but also to gain a better understanding of how Pygame works by implementing common game development concepts from scratch.

## Technologies Used

- Python 3
- Pygame

## Learning Objectives

This project was created to strengthen my understanding of:

- Event handling and user input
- Game loops
- Collision detection
- Object-oriented programming principles
- Managing multiple game objects with sprite groups
- Building a complete interactive application using Pygame

## Inspiration

Alien Invasion is inspired by classic arcade shooters such as **Space Invaders** and **Galaga**, recreating their core gameplay while serving as a practical introduction to game development with Python.
